== Description ==

This plugin adds an accessible "jump to content" bar, Shawn only when pressing on the TAB key. 
With this special bar, you will be able to allow disable people to "jump" to different content areas and pages, totally controlled by the admin panel.
This is plugin will assist you to farther accessible your website and make it easier for people with disabilities to find themselves in your site.

== Installation ==


1. Upload `zeevmobile folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. now you will see a button in your admin panel :"jump to content" where you can define the where the user will be jumped to.
4. that is it ! you can access your site, press TAB, and see the bar on the top of the screen.
